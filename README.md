# Magazine de Livres en Ligne

Bienvenue sur le repository de mon projet web ! 

# À propos du projet
Il s'agit d'une plateforme web simple et interactive sous forme de magazine en ligne dédié aux livres. 
Le concept est le suivant : l'utilisateur peut parcourir notre collection et, **en cliquant sur un livre spécifique, il accède directement à son résumé complet ainsi qu'à son prix.**

## Technologies utilisées
* **HTML** : Pour la structure du magazine.
* **CSS** : Pour le design, la mise en page et l'expérience utilisateur.
